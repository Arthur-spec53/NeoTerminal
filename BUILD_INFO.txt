🚀 快速部署命令
────────────────────────────────────────────────────────────────

1. 解压文件:
   tar -xzf xboard-frontend-dist.tar.gz

2. 复制到Web目录:
   cp -r dist/* /var/www/html/

3. 配置Nginx (示例):
   server {
       listen 80;
       root /var/www/html;
       index index.html;
       
       location / {
           try_files $uri $uri/ /index.html;
       }
       
       location /api {
           proxy_pass http://localhost:7001;
       }
   }

4. 重启服务:
   sudo nginx -s reload
